Ejercicios Nodos del DOOM

  1. Manipulación de Texto: Selecciona un elemento de texto en tu página y cambia su contenido utilizando JavaScript. Por ejemplo, cambia el texto de un párrafo o un encabezado.

    2. Encabezado original

    Parrafo original

  2. Agregar y Eliminar Elementos: Crea un nuevo elemento (por ejemplo, un párrafo o una lista) dinámicamente con JavaScript y agrégalo al DOM. Luego, elimina un elemento existente.

    3. Parrafo existente original

  3. Manipulación de Atributos: Selecciona un elemento que tenga un atributo (como un enlace con un atributo href) y cambia el valor del atributo utilizando JavaScript.
    4. Enlace
  4. Crear y Manipular Listas: Crea una lista (ul o ol) vacía en el DOM con JavaScript. Luego, agrega elementos a la lista dinámicamente.

    5. Lista de Elementos

  5. Comentarios Dinámicos: Utiliza JavaScript para agregar un comentario dinámico a tu código HTML. Por ejemplo, puedes agregar un comentario que indique la fecha y hora en que se realizó una acción específica.
  6. Cambiar Estilos Dinámicamente: Selecciona un elemento y cambia su estilo (por ejemplo, el color de fondo o el tamaño del texto) utilizando JavaScript en respuesta a algún evento, como un clic en un botón.
  7. Manejar Eventos: Asigna un manejador de eventos a un elemento (por ejemplo, un botón) y realiza alguna acción cuando ocurra el evento (por ejemplo, mostrar un mensaje en la consola cuando se haga clic en el botón).
  8. Aplicación de Lista de Tareas Interactiva: Crea una aplicación simple de lista de tareas. Los usuarios pueden agregar tareas, marcarlas como completadas y eliminarlas. Utiliza nodos de texto y elementos de lista.

    9. Lista de tareas

  9. Galería de Imágenes Dinámica: Crea una galería de imágenes en tu página. Utiliza JavaScript para cambiar dinámicamente las imágenes y sus descripciones cuando se hace clic en miniaturas.
  10. Actualización de Contenido en Tiempo Real: Utiliza JavaScript para actualizar dinámicamente el contenido de tu página en tiempo real. Esto podría ser información como la hora actual, la temperatura o cualquier dato que pueda cambiar.
